# 3. Product Overview

## What is DevFlow?

DevFlow is an AI-powered Developer Operating System.

Unlike traditional project management tools, DevFlow understands a
software project's progress, recommends the next engineering decision,
teaches concepts when needed, and guides developers from idea to
deployment.

## Tagline

**Build Software With Confidence.**

# 6. Learning Philosophy

This project is also a Software Engineering Bootcamp.

Every lesson follows this order:

1.  Concept
2.  Why
3.  History
4.  Real-world analogy
5.  Visual thinking
6.  Browser/runtime internals
7.  Build in DevFlow
8.  Best practices
9.  Code review
10. Interview questions
11. Assignment
12. CTO insight

Understanding always comes before implementation.

# 11. Working Agreement

We will not build DevFlow as a tutorial project.

We will build it as a real startup.

Every new feature must answer:

-   Does it reduce developer uncertainty?
-   Does it increase confidence?
-   Can we explain why it exists?

If the answer is no, we do not build it.
