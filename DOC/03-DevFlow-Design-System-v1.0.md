# DevFlow Design System

**Version:** 1.0\
**Status:** Draft

------------------------------------------------------------------------

# 1. Design Philosophy

DevFlow should feel calm, focused, professional, intelligent, and
premium. The interface should reduce stress rather than create it.

------------------------------------------------------------------------

# 2. Brand Personality

## We are

-   Calm
-   Supportive
-   Professional
-   Intelligent
-   Honest
-   Minimal

## We are not

-   Loud
-   Flashy
-   Distracting
-   Aggressive

------------------------------------------------------------------------

# 3. Emotional Goal

Every screen should guide users through this journey:

Confused → Clear → Confident → Productive

------------------------------------------------------------------------

# 4. Color System

  Role          Suggested Color   Purpose
  ------------- ----------------- -----------------------
  Background    Warm Ivory        Comfortable reading
  Primary       Sage Green        Growth & confidence
  Secondary     Soft Apricot      Positive actions
  Information   Dusty Blue        Guidance & trust
  Text          Slate Gray        Comfortable reading
  Success       Natural Green     Success feedback
  Warning       Soft Amber        Gentle warning
  Error         Muted Coral       Calm error indication

------------------------------------------------------------------------

# 5. Typography

Primary Font: Inter

Headings: - Bold - Clear - Confident

Body: - Easy to scan - Comfortable line height - High readability

------------------------------------------------------------------------

# 6. Spacing System

Use an 8px spacing scale.

4 8 16 24 32 48 64 96

------------------------------------------------------------------------

# 7. Components

Core UI components:

-   Buttons
-   Navigation
-   Hero
-   Cards
-   Inputs
-   AI Prompt
-   Mission Cards
-   Decision Cards
-   Learning Cards
-   Progress Cards
-   Footer

------------------------------------------------------------------------

# 8. Buttons

Primary - Filled - Sage Green

Secondary - Outline

Ghost - Transparent

Danger - Muted Coral

------------------------------------------------------------------------

# 9. Icons

Recommended: - Lucide Icons

Guidelines: - Rounded - Consistent stroke width - Minimal

------------------------------------------------------------------------

# 10. Animations

Purposeful only.

Duration: 150--250ms

Use for: - Hover - Focus - Loading - Progress

Avoid decorative animations.

------------------------------------------------------------------------

# 11. Accessibility

-   Semantic HTML
-   WCAG AA contrast
-   Keyboard navigation
-   Visible focus states
-   Meaningful labels

------------------------------------------------------------------------

# 12. Responsive Design

Support: - Desktop - Laptop - Tablet - Mobile

------------------------------------------------------------------------

# 13. Design Principles

Every component must answer: "Why does this exist?"

Every page must answer: "What should the user do next?"
